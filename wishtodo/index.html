# 功能简介

Wishtodo 是一个基于 JHipster 和 Spring 的微服务，实现了一个简单的 TODO 应用，同时支持 GraphQL 和 Restful 接口。

在线预览：

  • 接口服务 - Wishtodo Spring Service

仓库地址：

  • Service - codetrial/wish-todo-spring

后端主要技术：

  • Spring
  • JHipster
  • MySQL
  • GraphQL

# 系统划分

# Wisher

应用的用户。

# Todo

待办事项，必须属于某个用户。
