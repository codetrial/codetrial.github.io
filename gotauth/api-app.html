# 应用方接口

# 应用方管理

# GET /api/app

描述：查询应用方列表。支持分页及查询条件。

返回值：

{
  "status": "200",
  "message": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "code": "SystemX",
        "name": "The System X",
        "create_time": "2018-11-17T08:39:57.000Z",
        "update_time": "2018-11-17T08:41:09.000Z"
      }
    ],
    "pagination": {}
  }
}

# GET /api/app/:id

描述：查询单个应用方。

返回值：

{
  "status": "200",
  "message": "success",
  "data": {
    "id": 1,
    "code": "SystemX",
    "name": "The System X",
    "create_time": "2018-11-17T08:39:57.000Z",
    "update_time": "2018-11-17T08:41:09.000Z"
  }
}

# POST /api/app

描述：创建应用方。

返回值：

{
  "status": "200",
  "message": "success",
  "data": {
    "id": 1,
    "code": "SystemX",
    "name": "The System X",
    "create_time": "2018-11-17T08:39:57.000Z",
    "update_time": "2018-11-17T08:41:09.000Z"
  }
}

# PUT /api/app/:id

描述：修改应用方。

返回值：

{
  "status": "200",
  "message": "success",
  "data": [1]
}

# DELETE /api/app/:id

描述：删除应用方。会关联删除应用方下面的分组、角色和资源。

返回值：

{
  "status": "200",
  "message": "success",
  "data": [1]
}
